- **Direct Shopping Flow**: Removed intermediate modals for a faster "Add to Cart" experience directly from product cards.
- **Dynamic Shopping Cart**: Real-time updates to cart count and a floating cart modal to manage items.
- **Interactive Checkout**: A "Secure Checkout" section that dynamically mirrors the cart with subtotal and grand total calculations.
- **Live Location Integration**: Interactive Google Maps integration centered on Vijayawada, Andhra Pradesh.
- **Modern UI/UX**:
  - Responsive design using **Bootstrap 5.3.3**.
  - Smooth scroll-reveal animations.
  - Custom glassmorphism effects and premium typography (Outfit).
  - Real-time form validation for the checkout process.
- **Custom Toast Notifications**: Premium feedback system for user actions.
